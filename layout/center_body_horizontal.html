This demonstrates how to center align body contents horizontally. CSS is:

html {display:table; margin-left:auto; margin-right:auto; margin-top:20px; }
body {display:table-cell; text-align:center; }

Or can simplify margin specification to this:

html {display:table; margin:auto; margin-top:20px;  }
body {display:table-cell; text-align:center; }

However, a table still will not be centered:

Table not centered

To make a table also in the center, you need to include style="margin-left:auto; margin-right:auto;" in the table's attribute: <table style="margin-left:auto; margin-right:auto;">. Then you have this:
