Slider Library 1

  • This is a light-weighted, easy to use and configure image slider library. It's written in html/js/css and uses jquery.
  • Requirement: jquery 1.8 or above.
  • All the files are in ./js/slider/. See README.txt.
  • Assumption: banner image's width is 1000px. For different width, can adjust parameter in js file.
  • slider2.js supports 2 images. slider.js allows to add more images. In general, just use slider.js.
  • This does not support IE 8 and lower (disables slide animation for now).
  • Tested on Firefox, Chrome, Opera, Safari, IE 8.

Example Source Code

To use this slide library in your code, 1) copy and paste the above html code into your page, 2) add your images in div slide1_images, then 3) set parameters in js/slider/slider.js. The most important parameters in js/slider/slider.js include: a) the number of images. b) image width in pixels, each image is assumed to have the same width. c) automatic slide time interval (in ms. If <= 0, no automatic slide effect). d) whether show bullets.
