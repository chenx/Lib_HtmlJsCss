Demonstrate the MidiPlayer class: MidiPlayer.js

Created on: 4/1/2015
Last Modified: 4/2/2015

The MidiPlayer class is used to play midi by javascript, without any plugin. It requires a HTML5 browser: firefox, chrome, safari, opera, IE10+. Firefox works the best, followed by Chrome, Safari and Opera.

This depends on other 5 javascript files (audio.js, midifile.js, replayer.js, stream.js, synth.js) from [2], which is a demo of [1]. This is related to [3], which is a powerful tool to play midi in browser.

[1] http://matt.west.co.tt/music/jasmid-midi-synthesis-with-javascript-and-html5-audio/
[2] http://jsspeccy.zxdemo.org/jasmid/
[3] MIDI.js - Sequencing in Javascript.

Button is used here. You can also use div, span etc.
