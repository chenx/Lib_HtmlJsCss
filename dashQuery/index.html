Demonstrate the dashQuery javascript library, which is a minimum library written from scratch, whose grammar is similar to jQuery.

This helps to understand how jQuery is written. Although right now it passes id like _('id'), to use a syntax similar to jQuery: _('#id') or _('.class'), you just need to modify "function _(id)" to check the first letter of the given id string.

This library is from http://www.mikedoesweb.com/2012/creating-your-own-javascript-library/. The page is also downloaded locally and compressed to CreateYourOwnJSLib.zip.
